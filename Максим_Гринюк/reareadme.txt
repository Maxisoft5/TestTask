После распаковки проекта TrueCodeTask.zip для запуска необходима Visual Studio (2022) или Rider, MS Sql Server с localhost сервером.
После запуска проекта в IDE необходимо его собрать, далее в настройках проекта appsettings.Development.json в строке DefaultConnection можно поменять имя базы данных которая потом будет создана.
Далее можно или просто запустить проект TrueCodeTask с OpenAPI и таким образом запустить миграции или можно вручную, перейдя в консоль (если в Visual Studio 2022 то в Package Manager Console и в ней же выбрать Default Project - TrueCodeTask.DataAccess и ввести команду Update-Database для запуска миграций).

После запуска TrueCodeTask открывается доступ к API методам которые были описаны в задачах.
В TrueCodeTask.Tests написанны инграционные тесты по тем же задачам. Если необходимо проверить содержимое бд после запуска тестов, рекомендую закомментировать код удаляющий базу данных в тестах, но перед следующим запуском вручнуть удалить её.
